# Tev Nicolas - Career Site

Interactive career site for Tev Nicolas.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

```sh
npm install
npm run dev
```

## Editing Content

All content lives in `src/data/tev-profile.ts`. Edit that file to update experience, skills, chat responses, and fit assessments.

## Deploy

Connect this repo to Vercel. It auto-detects Vite and deploys on push.
